```C++
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int curMin = 1e5;
        int ans = 0, curMax = 0;
        for (auto p : prices)
        {
            if (p < curMin)
            {
                ans = max(ans, curMax - curMin);
                curMin = p;
                curMax = 0;
            }
            else if (p > curMax) curMax = p;
        }
        ans = max(ans, curMax - curMin);
        return ans;
    }
};
```
明显是没有从问题开始从简到难分析，导致选择了一条更复杂的路径：对于每一个位置i,记录位置i前面所有价格的最低价格即可（维护开销均摊O(1)）
